# Data Analytics Assignment 2 - Pune Zomato Analysis

## Overview
Analysis of Zomato restaurant data from four Pune locations: Baner, Bavdhan, Hinjawadi, and Wakad.

## Files
- `pune_zomato.ipynb` - Main analysis notebook
- `Z_Baner_cleaned.xlsx` - Baner location data
- `Z_Bavdhan_cleaned.xlsx` - Bavdhan location data
- `Z_Hinjawadi_cleaned.xlsx` - Hinjawadi location data
- `Z_Wakad_cleaned.xlsx` - Wakad location data

## Analysis Includes
- Data loading and merging
- Exploratory data analysis
- Location-wise comparisons
- Statistical analysis
- Visualizations
