# NYC Airbnb Data Analysis

This project analyzes the 2019 NYC Airbnb dataset to uncover insights about pricing, availability, and neighborhood trends.

## Dataset
- **Source**: AB_NYC_2019.csv
- **Records**: ~49,000 Airbnb listings
- **Time Period**: 2019 data from New York City

## Analysis Includes
- Data cleaning and preprocessing
- Statistical analysis and correlation studies  
- Price distribution by borough and room type
- Geographic visualization of listings
- Availability and review patterns

## Key Findings
1. Manhattan has the highest average prices
2. Entire homes are more expensive than private/shared rooms
3. Popular neighborhoods show higher competition
4. Location significantly impacts pricing strategy

## Files
- `airbnb_analysis.ipynb` - Main analysis notebook
- `AB_NYC_2019.csv` - Raw dataset
- `README.md` - Project documentation

## Libraries Used
- pandas
- matplotlib  
- seaborn

## How to Run
1. Ensure Python 3.x is installed
2. Install required libraries: `pip install pandas matplotlib seaborn`
3. Open `airbnb_analysis.ipynb` in Jupyter Notebook
4. Run all cells to generate analysis and visualizations
